\section{位相空間対}

位相幾何においては単独の位相空間$X$だけではなく，位相空間の対$(X,A)$が，
基本的な対象である．単独なものから，対を考えるということで，大きな
進歩がもたらされた．

\textbf{位相空間対}（topological pair）$(X,A)$とは，$X$が位相空間で，$A$がその部分空間であること
である．

2つの位相空間対$(X,A),(Y,B)$に対し，連続写像$f:	(X,A)	\to	(Y,B)$とは，
連続写像$f:X	\to	Y$で，
\[
		f(A)	\subset	B
\]
という条件を満たすものとする．

2つの位相空間対$(X,A)$と$(Y,B)$が位相同型であるとは，
単独の位相空間の場合と全く同様，連続写像$f	:	(X,	A)	\to	(Y,B)$と
$g	:	(Y,B)	\to	(X,	A)$を，合成$g	\circ	f	:	X	\to	X$，
$f	\circ	g	:	Y	\to	Y$が，ともにそれぞれ$X$，$Y$上の恒等写像に
等しくなるようのとることができることと定義する．

そのとき，制限$f	|_A	;	A	\to	B$と$g	|_B	:	B	\to	A$は，
ともに位相同型写像となる．

\例{1.10}
$(x,y,z)$空間$\R^3$で，単純にひもの両端をくっつけたものを$A$とする．空間$\R^3$で，ひもに，
結び目をつくってから，両端をくっつけたものを$B$とする（図\ref{fig1x2}）．
\begin{figure}[hbtp]
	\centering
	\includegraphics[bb=0 0 800 200,clip,width=98.2mm,height=49.7mm]{fig1x2.bmp}
	\caption{結び目}
	\label{fig1x2}
\end{figure}

そのとき，位相空間対$(\R^3,A)$と位相空間対$(\R^3,B)$は，決して位相同型にならないことが
証明できる（後で説明するホモトピー論を使い，補空間を基本群と考える）．
\□

2つの連続写像$f_i:	(	X	,A)	\to	(Y,B)	\:	(i=0,1)$がホモトピックであることの定義も，
前と同様に，$f_0$と$f_1$をつなぐ連続的に変化する連続写像の族
\[
	f_t	:	(X,A)	\to	(Y,B)	\:	(t	\in	[0,1])
\]
が存在することとする．

位相空間対$(X,A)$から$(Y,B)$への連続写像全体で，ホモトピックな連続写像は同じものと見なした集合を
\[
	[(X,A),(Y,B)]
\]
と書き，$(X,A)$から$(Y,B)$へのホモトピー集合という．$A	=	B	=	\varnothing$（空集合）
のとき，$(X,A),(Y,B)$をそれぞれ単に$X,Y$と書く．$[(X,\varnothing),(Y,\varnothing)]$は，
$X$から$Y$への連続写像全体で，ホモトピックな連続写像は同じものとみなした集合に等しいから
\[
	[X,Y]	=	[(X,\varnothing),(Y,\varnothing)]
\]
となる．

ホモトピー集合については，第3章で詳しく調べる．

\subsubsection{《 要約 》}
\begin{list}{}{}
	\item	\textbf{1.1}　位相同型とは，逆写像も連続な全単射写像が存在することである．
	\item \textbf{1.2}　ホモトピー同値とは，ホモトピーの意味で逆写像が存在することである．
	\item \textbf{1.3}　位相空間対の間の写像についても，位相同型，ホモトピー同値写像が同様に定まる．
\end{list}

\subsubsection{― 演習問題 ―}

\textbf{1.1}
アルファベットの\textsf{W}と\textsf{Z}が位相同型であることを示せ．

\textbf{1.2}
アルファベットの\textsf{P}と\textsf{R}がホモトピー同値であることを示せ．

\textbf{1.3}
アルファベットの\textsf{A}は，上部で$\Delta$を部分空間としているから，
$(\textsf{A},\Delta)$は，位相空間対である．また，アルファベットの\textsf{R}は，上部で\textsf{D}を
部分空間としているから，$(\textsf{R},\textsf{D})$も，位相空間対である．
$(\textsf{A},\Delta)$と$(\textsf{R},\textsf{D})$は，位相同型であることを示せ．

\chapter{理論の目的と概要}

まえがきで述べたように，位相幾何学は，つながっているか，離れているかという本質的な違い
のみをみつけていろいろな図形を分類する，数学の基本分野である．引っ張ったり，縮めたりするような
ささいな変化は無視し，図形のつながりだけを問題にする．

ものの長さを表わすのは何メートルという数であり，ものの重さを表すのも何グラムという単位のある
数である．しかし，図形のつながり具合を表すものを何とすればよいだろうか．1つの単位系で表すことの
できる数であろうか．

例えば，図形の穴の数を，1個、2個と数えるのは一つの表現であろう．
また穴とは何で，穴の数はどのように数えられるものであろうか．実は，それを数学的に説明するのが，
この本で紹介するホモトピー群，ホモロジー群，コホモロジー群であり，さらに，図形の曲がり具合の
程度を表わすものが特性類である．直観的に言えば，$i$次元のホモトピー群は，$i$次元の
“丸い穴”の様子を見ており，$i$次元のホモロジー群は，$i$次元の“部屋”の数を調べている
ということができるであろう．

このような一見とらえどころのない問題設定において，いかに現代数学が，
理論を厳密に作り上げ，群（ほとんどが，整数全体の足し算のなす群，または，それを mod で考える
巡回群）の言葉で，問題を代数的に整然と表すか味わって欲しい．図形を，つながりのみを基準に分類する
という問題で，本書を読む君達は，何をイメージできるか，少しの時間，思いを巡らせてみよう．そして，
読み終わった後に，今の想像を思い出して比較してほしい．それが，かけはなれていれば，君は，この本を
読んで新しい世界を開くことになる．それが，ほとんどの想像と違わないのならば，君は，
数学的センスが抜群なのだから，自身をもって先に進んでもらいたい．

目次を見れば内容の見当が，（知っている人は）つくだろうが，キーワードや，

\begin{list}{}{}
	\item	位相同型，ホモトピー同値，トーラス，M\"{o}bius の帯，
	\item 閉曲面，Klein のつぼ，セル複体，基本群，
	\item ホモトピー群，ホモロジー群，コホモロジー群，
	\item ファイバー束，ベクトル束，スペクトル系列，特性類，
\end{list}
などである．

これらの用語をどこかで見て，漠然とした興味を感じたことがあるのなら，
本書を読み終えた後，それらの定義がちっとも難しくなく，現代数学の
基本的な概念えあることが理解できるであろう．これらの用語に縁のなかった君も，
どうか，この単語たちの不思議な語感をいぶかり，本書を読み始める契機としてもらいたい．

位相幾何は，いくつかの魅力的な図形が，その理論の特徴的な例となるようにして
(意図的ではないのに），発展し続けてきた．それは，位相幾何のみならず，
数学の，あるいは学問全体の必然的な進歩の形態なのかもしれない．

本書の目標は，このような位相幾何の考え方に慣れ親しむことにある．ホモロジー群，
コホモロジー群の導入には，公理系を先に与えるが，読者の感覚的理解のためには，
この方法がよりよいのではなかろうか．具体的な構成は，
単体的なホモロジー群などで，後に与えられる．

もちろん，位相幾何は，位相空間論を基礎としており，
\begin{list}{}{}
	\item	砂田利一『曲面の幾何』(岩波書店），第2章
\end{list}
を読了しておれば，読者として理想であるが，位相空間論をなにも知らない読者にも理解できるように，
直観的な説明を行うことを心がけた．

また，本書を読むには，群論を知っていることが必要のように思うかもしてないが，本質的には，
次の2つの群を理解していればよい．

\begin{list}{}{}
	\item	（1）	整数の足し算，引き算がまた整数になるということ（これを，整数全体$\Z$が可換群をなすという）．
	\item	（2）	2つの整数が素数$p$の整数倍の差のとき，同じもと見なす（これを mod $p$で考えるという）．
整数全体を mod $p$で考えた集合$\Z_p$にも，そのまま足し算，引き算が定まる（これを，$\Z_p$が$p$次の巡回群をなすという）．
\end{list}

ただ一つ読者に求められる資質は，うわべを飾ったことばにだまされることのない，柔らかな伸びやかな感性である．
ともあれ，1章1章が，それぞれ友好的な気分で，君達に本質をつかまれることを期待して，装いを整えて待っている．
行動を起こすのは，今の君達である．
